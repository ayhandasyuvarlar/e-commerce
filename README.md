# E-Commerce

![img](./src/page/Admin/img/cute-business-woman-idea-thinking-present-pink-background-3d-rendering.jpg)

### Herkese merhaba bu projemde bir e-ticaret nasıl kurulur , basit bir e-ticaret sitesi kurarken nelere ihtiyaç duyarız bunları öğredim ve yaptım

### Proje içerisinde kullandığım kütüphaneler ve yapılar

- reset css
- chark ui
- styled
- antd
- react-router-dom
- moment
- axios
- react-query
- formik
- ProdectedRoute
- yup

### Proje Başlatmak için

```jsx
npm install
yarn
```

```jsx
npm run start

```
